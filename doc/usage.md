# llamune 操作説明書

このドキュメントでは、llamune の基本的な操作方法を説明します。
セットアップがまだの場合は [セットアップ手順](setup.md) を参照してください。

## 1. ログイン

ブラウザで http://localhost:5174 にアクセスします。

セットアップ時に `create_user.py` で作成したユーザー名とパスワードでログインしてください。

管理者ユーザーはすべての機能（プロジェクト・モデル・FTデータ・訓練ジョブの管理）が利用できます。一般ユーザーはチャットのみ利用可能です。

## 2. プロジェクトの作成

サイドバーの「プロジェクト」を開きます。

1. プロジェクト名を入力し「＋ 追加」をクリック
2. 作成されたプロジェクトにシステムプロンプトを設定

システムプロンプトはチャット時に LLM へ送信される指示文です。プロジェクトの用途に合わせて設定してください。

例：
```
必ず日本語で回答してください。
あなたはユーザーの業務を支援するアシスタントです。
```

## 3. モデルの登録

サイドバーの「モデル」を開きます。

### HuggingFace からモデルを検索・登録する

1. 「HuggingFace」タブをクリック
2. 検索ボックスにモデル名を入力して「検索」をクリック（例：`Gemma4`）
3. mlx-community の変換済みモデルが一覧表示されます（DL 数も確認できます）
4. 使用したいモデルをクリックするとファイルサイズが表示され、登録フォームに自動入力されます
5. 「＋ 登録」をクリック

> 💡 モデルのファイルサイズ ≒ メモリ使用量です。マシンの搭載メモリに合わせて選択してください。

### 対応モデル

llamune は以下の 2 つのバックエンドを自動判定して使い分けます。

| バックエンド | 対象モデル | 例 |
|---|---|---|
| mlx-lm | テキストモデル | Qwen2.5, Llama 等 |
| mlx-vlm | マルチモーダルモデル | Gemma4 等 |

判定はモデルの `config.json` 内の `model_type` に基づいて自動的に行われるため、ユーザーが意識する必要はありません。

## 4. チャット

サイドバーの「チャット」を開きます。

1. 「モデル選択」で登録済みモデルを選択
2. 「プロジェクト」で使用するプロジェクトを選択
3. メッセージを入力し、`Ctrl+Enter` で送信

初回のモデル選択時はモデルのダウンロードとロードが行われます（モデルサイズによって数分かかる場合があります）。

「会話をリセット」ボタンで会話履歴をクリアできます。
